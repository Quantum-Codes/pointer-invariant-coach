What this coaches

✓ in scope
  • Fast / slow pointer
  • Opposite-end two-pointer
  • Sliding window — fixed size
  • Sliding window — variable size
  • Partition pointer
— out of scope
  • Dynamic programming
  • Graph algorithms
  • Tree traversals
  • Pure greedy / math

If your problem doesn't have a pointer moving through an array while maintaining a condition — the coach will say so and stop. Specificity is the point.
